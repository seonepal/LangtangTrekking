Langtang Trekking - One Page Site
---------------------------------
Files:
- index.html : The one-page HTML file using Tailwind CDN for styling.

How to use:
1. Unzip the folder.
2. Open index.html in any modern browser.
3. Replace the background image URL or contact email with your own if desired.

Notes:
- This site uses Tailwind via CDN. For offline use, consider adding a local CSS build.
